# 今夜なに食べる？ ―― 夕ごはん献立決めアプリ

夕食の献立を迷わず決めるためのミニ Web アプリです。よく作る料理を登録しておき、「今日の気分」から逆引きして1品を提案します。決め方は3通り（気分でしぼる／ルーレットで引く／一覧からタップ）。

素の HTML / CSS / JavaScript だけで動きます。ビルドツール・npm 不要。

---

## 料理の足し方

普段さわるのは **`dishes.csv` の1ファイルだけ** です。1行追加すれば品が増えます。

```csv
name,category,moods,ingredients,note
豚しゃぶ,肉,さっぱり|夏向け,豚しゃぶ肉|ポン酢|レタス|きのこ|豆腐,
```

- `name`（料理名）だけ必須。`category`・`moods`・`ingredients`・`note` は空でも動きます。
- `moods`（気分タグ）は複数指定可。**区切りは半角パイプ `|`**（CSV の `,` と衝突させないため）。
- `ingredients`（主な材料）も複数指定可。**区切りは moods と同じ半角パイプ `|`**。買い物の確認用なので、**主な材料を5つくらいまで**を目安に書いてください（多くてもエラーにはなりません）。空欄でも構いません。
- 値の前後の空白・末尾カンマ・空行は自動で無視されます。
- 文字コードは **UTF-8** で保存してください（日本語が化けないように）。

### 材料の表示について

- ルーレットや一覧で選んだ1品は、**結果カードに材料が表示**されます（買い物前の確認用）。
- 候補リストの各行は、**右端のカートアイコンをタップすると材料が開きます**（その行の下に材料が並びます）。
- `ingredients` が空の品は、カートアイコン（材料の欄）じたいが表示されません。

### 生魚は入れない方針

2歳児が食べる前提のため、**刺身・海鮮丼などの生魚は登録しません**。料理を足すときもこの方針を守ってください。

---

## 気分タグ（チップ）の足し方

`moods` に新しい語を書けば、その語が**自動でチップに増えます**。固定リストはありません。

- チップの並びは推奨順（`がっつり, さっぱり, 鍋でラク, 中華, 子どもが喜ぶ, 早く完結, 夏向け`）を基準に、CSV に登場したものだけ表示されます。
- 推奨順にない新しいタグは末尾に追加されます。
- 先頭の「すべて」はアプリが自動でつける固定チップです。

## ローカルでの確認方法

`file://` で直接開くと CSV の読み込みに失敗するため、必ず簡易サーバ経由で開きます。

```bash
python3 -m http.server 8000
```

ブラウザで <http://localhost:8000> を開いてください。

> CSV が読めなかった場合でも白画面にはならず、アプリ内蔵の既定25品で表示し、上部に注意バーが出ます。

### 備考（note 欄のカンマ）

`note` 内にカンマを入れたいときは、`"..."` のようにダブルクオートで囲ってください（最低限のクオート対応をしています）。

## 公開方法の概略（GitHub Pages）

1. このリポジトリを GitHub に push する。
2. リポジトリの **Settings > Pages** で、公開ブランチ（例: `main` / ルート）を指定して有効化する。
3. 数分後に発行される `https://...github.io/...` の URL で公開されます。

詳しい手順はオーナーが実施してください。
